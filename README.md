# My Scripts

Это мой первый репозиторий с полезными скриптами для пентестера.

## Скрипты

- `link_checker.py` — проверяет все ссылки на сайте и сообщает, какие из них рабочие, а какие нет.
- `Email scraper.py` — собирает email-адреса с указанных страниц сайта.
- `Port scanner.py` — проверяет открытые порты на указанном хосте.
- `Robots chechker.py` — проверяет наличие и содержимое `robots.txt`.
- `first_script.py` — мой первый скрипт, для истории.
